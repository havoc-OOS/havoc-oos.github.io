Installation Instructions

Follow these steps to install havocOOS on your device

1

Prerequisites

  • Unlocked bootloader
  • Custom recovery (TWRP/OrangeFox)
  • Latest firmware installed
  • Backup all important data
2

Download Required Files

  • Download havocOOS ROM zip
  • Download GApps (optional)
  • Download Magisk (optional, for root)
3

Boot to Recovery

  • Power off your device
  • Hold Volume Up + Power button
  • Release when you see recovery logo
4

Wipe Data

  • In recovery, go to "Wipe"
  • Select "Format Data"
  • Type "yes" and confirm
5

Flash ROM

  • Go to "Install" and select ROM zip
  • Swipe to flash
  • Flash GApps if desired
  • Flash Magisk if you want root
6

Reboot

  • Go back to main menu
  • Select "Reboot" â†’ "System"
  • Wait for first boot (may take 5-10 minutes)
